Purpose

The purpose of this project was to allow me to simply find the most accurate pair of resistors for a needed purpose. Often when selecting resistor values, there is a sacrafice when using a single value as they can never be as accurate as needed. This project allows me to find the nearest possible pair of values either in series or parallel

What did I learn?

Through this project I learned that it can be very simple to find the required values when the correct equipment is used. Instead of selecting values for an hour and trying to find the nearest combination, I can simply use a nested for loop to find the closest pair in less than a second.

Future inspiration

Future inspiration for this project is to convert the python script to javascript so that I am able to display and use the caluclator on this page here. I also plan to add a greater number of resistor series, allowing the calulcator to be used for more purposes and by more people.
