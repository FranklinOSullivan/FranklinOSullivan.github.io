Purpose

The purpose of this project was to ensure that I was still able to perform simple Javascript calculations and function calls. Throughout a few iterations of my calculator this became better and much more second nature again.

Execution

I decided that I would begin by developing a very simple calculator to ensure that I was using Javascript correctly before switching to something more difficult.

This first iteration of my calculator was very simple and involved only a few input boxes and buttons. This calculator was able to do the most simple math and not much more.

To continue this idea, I found it might be useful to utilize a table to arrange the numbers and modifiers into a more recogniseable calculator shape. This meant that there would more buttons and no more input boxes, leading to more Javascript.

 

This execution of the calculator allows for much more familiar use and is very intuitive to use. Although some more CSS could easily be added to improve the look of the calucltor, I found that this would not be nessecary and could be skipped.

What did I learn?

Throughout this project, I learned how easy and powerful Javascript is to use. I also got a chance to do some simple math and work with tables. This helped me to understand positioning better and helped solidify my understanding of how useful a table can be in HTML as well as how versitile they are.

Future inspiration

Although I doubt that I will ever need to recreate this project, I think that it can be a very useful inspiration for far more specific calculators. This project could easily be used to help solve problems that come up often in my classes at uni. (Maybe something like finding the resistor values to set a certain amount of gain for an OPAMP). This would be very simple to implement and would likely help me quite often as it will save a great amount of time.
